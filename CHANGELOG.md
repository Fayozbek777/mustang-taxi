# Журнал изменений (Changelog)

Все заметные изменения в этом проекте будут документироваться в данном файле.

Этот проект следует правилам семантического версионирования ([SemVer](https://semver.org)). Формат основан на стандартах [Keep a Changelog](https://keepachangelog.com).

---

## — 2026-06-09

### ✨ Добавлено (Added)

- **Интеграция TinyPNG API:** Реализован сквозной пайплайн автоматического сжатия и оптимизации графических ассетов «на лету» при добавлении новых продуктов (`Add Product`) через административную панель. Уменьшает размер входящих изображений до 75% без потери визуального качества.
- **Мониторинг Web-Vitals:** Интегрирована библиотека `web-vitals` (^5.3.0) для непрерывного отслеживания метрик производительности интерфейса (LCP, FID, CLS) в реальном времени.
- **Мониторинг Stats.js:** Интегрирован модуль `stats.js` (^0.17.0) для реалтайм-аудита частоты кадров (FPS), задержки ввода и времени рендеринга сложных DOM-деревьев.
- Добавлен новый уровень абстракции для логирования сборки и автоматической валидации конфигурации окружения во время деплоя.

### 🚀 Улучшено (Improved)

- **Performance Optimization:** Проведена комплексная оптимизация клиентской части, сокращено время интерактивности (Time to Interactive) и достигнут целевой показатель Lighthouse Score 100/100.
- **Апгрейд Админки:** Существенно переработан интерфейс и UX панели управления автопарком; ускорена реактивность переключения состояний системы.
- Оптимизированы заголовки кэширования для статических ассетов в конфигурации `vercel.json`.

## — 2026-03-31

### 🔐 Изменено (Changed)

- **Переход на DB-less Архитектуру:** Полный отказ от внешних реляционных баз данных. Архитектура переведена на кастомный реактивный движок на базе Python Flask и транзакционное хранилище GitHub REST API.

### 🛡️ Безопасность (Security)

- **Security Path Isolation:** Логика входа в панель администратора полностью изолирована и скрыта за динамически вычисляемой криптографической переменной окружения `${VITE_ADMIN_PATH}` для предотвращения brute-force атак.
