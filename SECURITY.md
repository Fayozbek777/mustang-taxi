# Security Policy (Bug Bounty Program)

## Supported Versions

Only the latest production releases hosted on Vercel are supported for security audits.

| Version | Supported |
| ------- | --------- |
| 2.0.x   | 🟢 Yes    |
| 1.5.x   | ❌ No     |

## Reporting a Vulnerability

If you discover any critical vulnerability regarding the hidden route (`${VITE_ADMIN_PATH}`) or the DB-less GitHub REST API Engine, please **DO NOT** open a public issue.

Report vulnerabilities securely by contacting the lead architects via:

- **Telegram Secure Chat:** [@YandexMustang](https://t.me/YandexMustang)

### Bug Bounty Rewards

Valid critical exploits affecting user billing data or system takeover are eligible for a private bounty reward up to **$5,000 USD** (sponsored by internal corporate budgets).
